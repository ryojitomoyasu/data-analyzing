# トラブルシューティングガイド

アプリケーション（BizStrategy AI Partner）が起動しない、または動作がおかしい場合の対処法です。

## 1. アプリが立ち上がらない場合

### 症状
- 黒い画面（コマンドプロンプト）が一瞬で消える
- 「Port 8501 is already in use」というエラーが出る
- ブラウザに「このサイトにアクセスできません」と表示される

### 解決策：強制起動ツールの使用
同じフォルダにある **`force_start.bat`** をダブルクリックして実行してください。
このツールは以下の処理を自動で行います：
1. バックグラウンドで残っている古いプロセスを強制終了
2. 必要なライブラリがインストールされているかチェック
3. アプリを再起動

## 2. Gemini AI が反応しない場合

### 症状
- チャットでエラーが出る
- 「API Key Error」と表示される

### 解決策
1. サイドバーの「Gemini API Key」欄に正しいキーが入力されているか確認してください。
2. `.env` ファイルに `GOOGLE_API_KEY=あなたのキー` が正しく記述されているか確認してください。

## 3. データの読み込みでエラーが出る場合

### 症状
- ファイルをアップロードすると赤枠のエラーが出る

### 解決策
- ファイル形式が `.xlsx` または `.csv` であることを確認してください。
- 列名に「売上」「日付」「カテゴリ」などのキーワードが含まれているか確認してください。
- サンプルデータ（`dummy_sales.csv`）が正常に読み込めるか試してください。

---
上記で解決しない場合は、Antigravity（AIアシスタント）にエラーメッセージを伝えて相談してください。
